## Basic Functions
This app shows the basic usage of the following components:

1. Mahapps Library (Hamburger Menu, Buttons, Table, Checkbox, )
2. SQLite Library
3. Custom Core DLL (SQLiteHandler, QueryBuilder)
4. Basic Unit Testing with Nunit
